# WHALE 🐋
**W**elfare, **H**ealthcare, **A**dministration, **L**ong-term care, and **E**ducation

福祉医療介護統合管理システム

## 概要
WHALEは福祉・医療・行政・介護・教育分野における包括的な情報管理システムです。利用者の日常的な健康記録から職員の業務管理まで、一元的に管理できるWebアプリケーションです。

## 主な機能

### 🏠 利用者向け機能
- **日次記録入力**
  - 睡眠時間・質の記録
  - 食事摂取状況の管理
  - 服薬記録
  - バイタルサイン（体温・血圧・脈拍・体重）
  - 活動記録（運動・入浴・身だしなみ）
  - 気分・心理状態の記録
  - 自由記述（感想・相談・心配事など）

- **データ管理・分析**
  - 記録履歴の閲覧・検索
  - 統計情報の表示（平均気分スコア、服薬率、連続記録日数等）
  - データのエクスポート（CSV、JSON、PDF形式）
  - 印刷機能

- **プロフィール管理**
  - 個人情報の登録・更新
  - 緊急連絡先の管理

### 👨‍⚕️ 職員向け機能
- **利用者管理**
  - 全利用者の記録状況一覧表示
  - アラート・要注意事項の確認
  - 利用者情報の登録・編集・削除
  - パスワードリセット機能

- **職員管理**
  - 職員情報の登録・編集
  - 職位・資格管理
  - 勤務状況管理
  - 権限設定

- **レポート・統計**
  - 期間別集計レポート
  - バイタルサイン推移
  - 服薬状況分析
  - 気分・心理状態の変遷
  - アラート履歴

- **システム管理**
  - データバックアップ・復元
  - アラート閾値設定
  - 通知設定
  - 表示設定

## 使用方法

### 1. システムにアクセス
[WHALE](https://uchida16104.github.io/WHALE/)


### 2. ログイン
- 利用者または職員のIDとパスワードでログイン
- デモアカウントで機能を体験可能

### 3. 記録入力（利用者）
- 「今日の記録を開始する」をクリック
- 各項目を入力（進捗バーで入力状況を確認）
- 「記録を保存」で保存完了

### 4. 管理・確認（職員）
- ダッシュボードで全体状況を把握
- アラートや要注意事項を確認
- 各種管理画面で詳細な操作を実行

### 💾 データの永続性について

- ブラウザのデータを削除すると記録が消失します
- 定期的なバックアップ（エクスポート）を推奨します
